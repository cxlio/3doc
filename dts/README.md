# @cxl/dts 
	
[![npm version](https://badge.fury.io/js/%40cxl%2Fdts.svg)](https://badge.fury.io/js/%40cxl%2Fdts)

Generate a JSON AST of your TypeScript public API (exports, types, JSDoc) from tsconfig or source, resolving symbols and project references for documentation.
